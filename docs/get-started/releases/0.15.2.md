  ([PR #2346](https://github.com/NVIDIA/Megatron-LM/pull/2346)).

  ([PR #2575](https://github.com/NVIDIA/Megatron-LM/pull/2575)).

  ([PR #2471](https://github.com/NVIDIA/Megatron-LM/pull/2471)).
