## Description: <br>
Split a PR into multiple PRs to reduce the number of required CODEOWNERS reviewer groups. <br>

This skill is ready for commercial/non-commercial use. <br>

## Owner: NVIDIA <br>

## Use Case: <br>
Developers and engineers use this skill to break large pull requests into smaller, focused PRs that each require fewer CODEOWNERS reviewer groups, reducing review burden and accelerating merge cycles. <br>
